-----

### Requirements

- PHP 7.1.0+

### Installation

The only supported way of installing this package is using [Composer](http://getcomposer.org).

- Add `nyx/connect` as a dependency to your project's `composer.json` file.
- Run `composer update` and you're ready to go.
